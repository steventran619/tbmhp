# TBMHP Client
Preview Link:

https://steventran619.github.io/tbmhp/

## To get started run the following:
```
cd client
npm install
npm run dev
```

## Build Locally
```
npm run build
```

## Test Build Locally
```
npm run preview
```

## Deploy to GitHub Pages
**WARNING this will replace the deployed site with the latest local build**
```
npm run deploy
```
